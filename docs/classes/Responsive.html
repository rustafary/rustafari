Class Responsive

A utility class that enhances the responsiveness of web applications. It provides callbacks for different breakpoints and allows dynamic adjustment of styles based on screen size.

Usage example:

const responsive = new Responsive();
responsive.on(
  () => console.log("Extra-small screen"),
  () => console.log("Large screen"),
  () => console.log("Extra-large screen"),
  () => console.log("Extra-extra-large screen")
);

applyResponsiveClasses

  • applyResponsiveClasses(element, classes): void

  • Adds responsive classes to an element based on the current breakpoint. Changes the class names of an element dynamically as the screen size changes.

    Parameters

    • element: HTMLElement

      The target HTML element.

    • classes: {
          lg?: string;
          xl?: string;
          xs?: string;
          xxl?: string;
      }

      An object containing class names for each breakpoint.

      • Optionallg?: string

        Class name for the large breakpoint.

      • Optionalxl?: string

        Class name for the extra-large breakpoint.

      • Optionalxs?: string

        Class name for the extra-small breakpoint.

      • Optionalxxl?: string

        Class name for the extra-extra-large breakpoint.

    Returns void

    Example

    const responsive = new Responsive();
    const element = document.getElementById("myElement");
    responsive.applyResponsiveClasses(element!, {
      xs: "mobile-class",
      lg: "tablet-class",
      xl: "desktop-class",
      xxl: "large-desktop-class"
    });
